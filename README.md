# Beauty_rc

## サイト概要
サロンへ行った際にどんな事をしたかを記録できます。
日付順に写真や内容、金額をを投稿し過去を振り返ることが出来ます。

### サイトテーマ
自分専用投稿サイト

### テーマを選んだ理由
自分が美容室へ行った際、前回どんな髪型にしたか忘れてしまったのがきっかけで、自分用のメモとしてそんなサイトがあれば便利だと思い選びました。

### ターゲットユーザ
サロンへ通う男性、女性向け
